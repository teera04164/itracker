PNG, JPG, WEBP — อัพโหลดผ่าน Cloudinary

คลิกเพื่อเปลี่ยนรูป

⚙️ ตั้งค่า Cloudinary (ครั้งเดียว)
วิธีสร้าง Unsigned Preset:
1. เข้า cloudinary.com → Settings → Upload
2. เลื่อนลงหา "Upload presets" → Add upload preset
3. เปลี่ยน Signing mode เป็น Unsigned
4. บันทึก → คัดลอก Preset name
5. Cloud Name อยู่ที่มุมบนขวาของ Dashboard

⚠️ ถ้าไม่ตั้งค่า Cloudinary ระบบจะบันทึกรูปเป็น Base64 ใน localStorage แทน (ใช้งานได้ปกติ แต่กินพื้นที่มากกว่า)

📊 Google Sheet (ตั้งค่าครั้งเดียว)
1. เปิด script.google.com → New project
2. วางโค้ด Apps Script ที่ได้รับ → Save
3. Deploy → New deployment → Web app
4. Execute as: Me | Access: Anyone
5. คัดลอก Web app URL มาวางด้านล่าง
